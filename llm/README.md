Initially

## Set up the demo

 - Set up virtual environment
    ```bash
    cd llm
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```
 - Create .env file under `llm` directory and copy `API_TOKEN` into it.
 - `python llm_request.py`
